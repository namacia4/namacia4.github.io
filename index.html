Tableau Project-
Airbnb in Chicago

Description: Conducted a comprehensive analysis of Airbnb data for the Chicago area spanning the period from 2023 to 2024. The primary goal was to transform the raw data into a meaningful, interactive dashboard for enhanced insights and visualization.

Key Responsibilities and Achievements:
Data Enhancement: Faced with limited information, the project involved manually correlating each of Chicago's 77 community areas with their respective zip codes. Employed Excel's IFS() formula to efficiently match community areas to the corresponding zip codes, overcoming the absence of direct zip code data in the original dataset.
Data Analysis: Performed an in-depth analysis of the Airbnb dataset, exploring various dimensions such as pricing trends. Utilized statistical methods to identify patterns and outliers.
Dashboard Creation: Developed an interactive dashboard using Tableau to present the analyzed data in a user-friendly and insightful manner. Integrated charts, graphs, and filters to facilitate dynamic exploration of the Airbnb trends across different community areas.
User Experience Enhancement: Ensured a seamless user experience by incorporating intuitive design elements and interactive features within the dashboard. This allows users to easily navigate through the data and extract relevant information.
Visualizations: Created visualizations highlighting key insights, including popular neighborhoods. These visualizations serve as valuable tools for stakeholders to make informed decisions.
Technologies Used: Microsoft Excel for data manipulation and preprocessing, Tableau for creating interactive dashboards.

Outcome: The resulting interactive Airbnb Data Analysis Dashboard provides a comprehensive overview of the 2023-2024 trends in the Chicago market. Stakeholders can now effortlessly explore and understand the dynamics of Airbnb listings across different community areas, enabling data-driven decision-making for various purposes, including real estate investment and tourism planning.

    Web Scraping with BeautifulSoup: Employed the BeautifulSoup library to parse and navigate the HTML content. Utilized the find_all method to locate and extract the desired table from the HTML structure.
    Data Formatting: Implemented data formatting procedures to clean and structure the extracted tabular data. Addressed issues such as unwanted characters, missing values, or irregularities within the data to ensure uniformity.
    Pandas for Data Manipulation: Utilized the pandas library to convert the formatted data into a DataFrame, allowing for further manipulation and analysis. Leveraged pandas functionalities for filtering, sorting, or aggregating the data as needed.
    Export to CSV: Exported the formatted data to a CSV file using pandas. Ensured that the CSV file captured the cleaned and structured tabular data, making it easily shareable and compatible with other data analysis tools.
    Technologies Used: Python for scripting and implementation. Libraries: requests, BeautifulSoup, pandas.

    Outcome: The Web Scraping and Data Formatting project successfully demonstrated the use of Python libraries for extracting, cleaning, and exporting data from a Wikipedia page. The resulting CSV file encapsulated the formatted data, providing a structured dataset for further analysis. This project serves as a practical example of utilizing web scraping techniques to extract valuable information from online sources.
